# Audio

Intro amplop sekarang memakai musik original procedural dari Web Audio API, jadi tidak membutuhkan file audio.

Jika suatu saat ingin mengganti dengan file lokal, letakkan MP3 di folder ini dan isi `bgm` pada `../font/content.js`, misalnya:

```js
bgm: "audio/namafile.mp3"
```

Gunakan hanya audio yang kamu punya hak untuk pakai.
