# Envelope Intro

Folder ini berisi animasi amplop pembuka yang diadaptasi dari:

https://github.com/Genius-Society/eLuvLetter

File entry lokal untuk animasi adalah:

`envelope.html`

Halaman utama proyek tetap berada di:

`../index.html`

## Struktur

- `envelope.html` - halaman animasi amplop yang dimuat lewat iframe dari halaman utama.
- `css/` - style amplop, background ulang tahun, typing, dan musik.
- `js/` - script animasi amplop, typing, background birthday, dan helper lokal.
- `img/` - aset visual amplop dari repo sumber.
- `font/` - font/aset konten lokal untuk isi surat.
- `audio/` - tempat file musik lokal opsional; default memakai musik original procedural.
- `LICENSE` - lisensi Apache-2.0 dari repo sumber.

## Catatan

Nama file `index.html` di folder ini sengaja diganti menjadi `envelope.html` supaya tidak tertukar dengan `../index.html`, yang merupakan halaman utama website.
